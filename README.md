# ETL Seguros Pipeline

Este proyecto implementa un pipeline ETL para procesar datasets de clientes, pólizas y siniestros.

## Estructura
- `data/curated/`: datasets finales limpios
- `data/rejects/`: registros descartados
- `etl_pipeline.ipynb`: notebook con el código

## Ejecución
1. Clonar este repositorio.
2. Abrir `etl_pipeline.ipynb` en Google Colab o Jupyter.
3. Ejecutar las celdas en orden.
4. Los resultados se guardan en `data/curated` y `data/rejects`.
